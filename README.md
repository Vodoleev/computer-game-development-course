## Описание проекта

3D-раннер от третьего лица: игрок управляет кубом, который бежит по бесконечной дорожке с тремя полосами.  
Нужно уклоняться от препятствий, собирать бонусы и выживать как можно дольше, пока скорость и сложность растут.

## Управление

- Движение по полосам:
  - `←` / `→` или `A` / `D`
- Прыжок:
  - `Space`, `↑`
- Игра стартует автоматически при нажатии кнопки Play в Unity.

## Реализованный функционал

### Игрок
- Автоматическое движение вперёд с настраиваемой скоростью.
- Три полосы движения (левая, центральная, правая), плавное перестроение.
- Прыжок (с ограничением по количеству прыжков подряд).
- Система здоровья:
  - начальное здоровье `maxHealth`;
  - получение урона при столкновении с препятствиями;
  - лечение через бонусы;
  - смерть и автоматический рестарт сцены при 0 HP.
- Управление построено на **новой Input System**.

### Препятствия
- Два типа препятствий с разным внешним видом и уроном:
  - большой куб;
  - маленький цилиндр.
- Препятствия спавнятся:
  - с фиксированным интервалом;
  - в случайной полосе;
  - на случайном расстоянии вперёд по оси Z.
- Старые препятствия автоматически удаляются, когда оказываются далеко позади игрока.

### Бонусы
- Два типа бонусов:
  - `HealBonus` — восстанавливает здоровье игрока.
  - `SpeedBonus` — временно ускоряет движение.
- Бонусы автоматически спавнятся независимо от препятствий, в случайных полосах.
- На экране отображаются текстовые уведомления при подборе бонусов.

### Ускорение игры
- Скрипт `DifficultyScaler`:
  - постепенно увеличивает скорость движения игрока;
  - уменьшает интервал спавна препятствий до минимального значения.
- Текущая скорость игрока выводится на экран.

### Бесконечный мир
- Дорожка реализована с помощью двух повторяющихся плиток.
- При удалении игрока от плитки она перекидывается вперёд, за счёт чего создаётся бесконечная дорожка.

### UI
- Полоска здоровья (Health Bar), отображающая текущее HP.
- Текст с текущей скоростью.
- Текстовые сообщения о полученных бонусах.

## Структура проекта

Основные скрипты:

- `PlayerController` — движение, полосы, прыжок, применение ускорения.
- `PlayerHealth` — здоровье, урон, лечение, смерть и рестарт.
- `Obstacle` — логика препятствий (урон игроку, удаление позади).
- `ObstacleSpawner` — спавн препятствий и бонусов.
- `BonusBase`, `HealBonus`, `SpeedBonus` — система бонусов.
- `DifficultyScaler` — рост сложности (скорость + частота препятствий).
- `GroundLooper` — бесконечная дорожка.
- `UIManager` — управление UI (здоровье, скорость, сообщения).
